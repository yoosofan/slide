    * State machine needs memory

* checks a given source program
* creates the parse tree
* Otherwise gives the error messages
* → , λ , |
* S → aA
* S → b

    * -> instead of →

    * S → (S)
    * S → a

    * S → [A]
    * A → aS
    * A → λ
    * [], [a[]], [a[a[]]]

Grammar of simpler calculator
================================
#. It has just ``+, *, (, )``
#. There is no priority
#. No unary operator (+)
#. Use paranthesis for more that one operator
    #. (2+3)+4
    #. 2+(4+3)
    #. (2+(4+3))+5
    #. (2*(4*3))*5
#. Terminals
#. INT (for simplicity we use token.type: i)
    #. (i+i)+i
    #. i+(i+i)
    #. (i+(i+i))+i
    #. (i*(i*i))*i
#. {'+', '*'}
#. {'(', ')'}
#. No need for naming operators, just using the symbol
#. Use recursion

====================
.. container::

  #. S → A + A
  #. S → A * A
  #. S → A
  #. A → (S)
  #. A → i

  * 43 + 87

  .. class:: substejp

    #. S ⇒ A + A ⇒

Derivation Tree(III)
====================
.. container::

  * S → A + A | A * A | A
  * A → (S) | i

  * 3 * (32 + 5)

  .. class:: substep

    Left Most Derivation

    #. S ⇒ A * A ⇒
    #. i[3] * A ⇒
    #. i[3] * (S) ⇒
    #. i[3] * ( A + A ) ⇒
    #. i[3] * ( i[32] + A ) ⇒
    #. i[3] * ( i[32] + i[5] )

    Right Most Derivation

    #. S ⇒ A * A ⇒
    #. A * ( S ) ⇒
    #. A * ( A + A ) ⇒
    #. A * ( A + i[5] ) ⇒
    #. A * ( i[32] + i[5] ) ⇒
    #. i[3] * ( i[32] + i[5] )
