First set

  1. If X → λ is a production rule then λ ∈ first(X)

  2. If X → Y 1 Y 2 . . . . Y n is a production rule then

    1. first( Y 1 ) ⊂ first(X)

    2. first( Y 2 ) ⊂ first(X) if λ ∈ first( Y 1 ) or Y 1 ⇒ * λ

    3. first( Y 3 ) ⊂ first(X) if λ ∈ first( Y 1 ) and λ ∈ first( Y 2 )

    4. first( Y i ) ⊂ first(X) if λ ∈ first( Y j ) for j = 1, 2, 3, i-1

    5. λ ∈ first(X) if λ ∈ first( Y i ) for i = 1, 2, 3, n or X ⇒ * λ

Follow Set(II)

  1. If S is start symbol then $ ∈ follow(S)

  2. If X → α Y then follow(X) ⊂ follow(Y)

  3. If X → Y 1 Y 2 Y 3 Y 4 . . . . . Y n is a production rule then
    1. (first( Y 3 ) - {λ} ) ⊂ follow( Y 2 )

      • if M → α A a β is a production rule then a ∈ follow(A)

    2. If λ ∈ first( Y 3 ) then (first( Y 4 ) - {λ} ) ⊂ follow( Y 2 )

      • or (first( Y 3 ) ∪ first( Y 4 ) - {λ} ) ⊂ follow( Y 2 )

    3. If λ ∈ first( Y j ) for j = 3,4, ....., i-1 then (first( Y i ) - {λ} ) ⊂ follow( Y 2 )

    4. If λ ∈ first( Y j ) for j = 3,4, ....., n then follow(X) ⊂ follow( Y 2 )
