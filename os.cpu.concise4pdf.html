Estimating Service Time(I)

  1. τ n = t 0 + t 1 + t 2 + . . . + t n − 1 n
  2. n * τ n = t 0 + t 1 + t 2 + . . . + t n − 1
  3. τ n + 1 = t 0 + t 1 + t 2 + . . . + t n − 1 + t n n + 1
  4. = t 0 + t 1 + t 2 + . . . + t n − 1 n + 1 + t n n + 1
  5. τ n + 1 = n * τ n n + 1 + t n n + 1
  6. τ n + 1 = n n + 1 * τ n + 1 n + 1 * t n
11

Estimating Service Time(II)

  1. τ n + 1 = n n + 1 * τ n + 1 n + 1 * t n
  2. τ n + 1 = n + 1 − 1 n + 1 * τ n + 1 n + 1 * t n
  3. τ n + 1 = ( n + 1 n + 1 − 1 n + 1 ) * τ n + 1 n + 1 * t n
  4. τ n + 1 = ( 1 − 1 n + 1 ) * τ n + 1 n + 1 * t n
  5. α = 1 n + 1
    τ n + 1 = ( 1 − α ) * τ n + α * t n
12

Estimating Service Time(III)

  1. α = 1 n + 1 , τ n + 1 = ( 1 − α ) * τ n + α * t n
  2. t n = a c t u a l l e n g t h o f n t h s e r v i c e t i m e
  3. τ n + 1 = p r e d i c t e d v a l u e f o r t h e n e x t s e r v i c e t i m e
  4. 0 ≼ α ≼ 1 , τ n + 1 = ( 1 − α ) * τ n + α * t n
  5. α → 0
